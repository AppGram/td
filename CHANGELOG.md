# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-19

### Added
- Initial release
- Beautiful TUI with multiple color schemes (default, black, copper, seafoam)
- Workspace management for organizing tasks
- Inline task syntax for quick entry (`#tags`, `@dates`, `!priority`)
- Smart dashboard with progress stats, today's tasks, overdue count
- Details panel showing task metadata (toggle with `m`)
- Subtask support with indentation (`>` / `<`)
- Vim-style navigation (`j/k`, `gg`, `G`)
- Command mode for task metadata editing (`:due`, `:tag`, `:priority`, `:clear`)
- Date shortcuts (today, tomorrow, weekdays)
- Task completion toggle
- Search/filter functionality
- ASCII art header with customization
- Weather widget (optional)
- SQLite database storage
- Cross-platform support (macOS, Linux, Windows)
